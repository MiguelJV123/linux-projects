# Project 3 – Terminal Todo / Note Manager

## Description
A simple terminal-based task and note manager.  
This project introduces user input handling, file storage, menus, and basic scripting logic.  
It demonstrates Linux scripting as a way to create your own custom tools.

## Files
- `todo.sh` or `todo.py` – Main application script
- `todo_data.txt` – Stores tasks/notes persistently

## Requirements
- Linux OS
- Bash **or** Python 3 (depending on the version you made)

## Features
- Add tasks
- View tasks
- Delete completed tasks
- Persistent storage (data saved even after shutdown)
- Simple menu-driven interface

### Make script executable

chmod +x todo.sh
